## soOot!
`golang`, `bootstrap`, `postgreSQL`로 개발된 대학 강의 평가 및 정보 공유 서비스 sooot!입니다.
수강신청시 강의에 대한 정보를 알 방법이 없어 답답한 마음에서 개발을 시작했습니다.
